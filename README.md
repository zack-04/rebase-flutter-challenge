# Rebase Challenge: Flutter (Stage 0)

Fix a simple Counter App.

### Time Limit
This should take no more than **30 minutes**.

### The Task
1.  **Clone** the code.
2.  **Fix the State Logic**:
    *   The "+" button doesn't seem to do anything. Find out why `setState` isn't updating the UI correctly.
3.  **Host it**:
    *   Build for Web and host on GitHub Pages or Netlify.
4.  **Submit**:
    *   Paste your **GitHub Repository Link** and **Live Website Link** in the Rebase LMS dashboard.
